Une page dans la tradition uses.tech : ce avec quoi je travaille au quotidien.

OS & Environnement #

Ubuntu 24.04 LTS — ma machine principale, full Linux. Windows 11 sur une partition dédiée uniquement pour le jeu.

Terminal — le terminal natif Ubuntu avec Bash. Simple, universel, cohérent avec les environnements de prod.

VSCode — éditeur principal, avec les extensions Kubernetes, Docker, GitLens et HashiCorp Terraform/OpenTofu.

Infrastructure locale #

K3S — cluster Kubernetes local léger pour les POCs, la veille techno et tester des charts Helm avant de les pousser en prod. Démarrage en quelques secondes, footprint minimal.

nerdctl — CLI Docker-compatible par-dessus containerd, pour builder et manipuler des images sans démon Docker. Utilisé au quotidien pour le build local.

CLI du quotidien #

kubectl — incontournable. Avec les alias classiques (k, kgp, kgn…) dans le .bashrc.

Helm — gestion des déploiements sur K3S local et les clusters de prod.

OpenTofu — pour tout ce qui touche à l’infrastructure as code.

git — avec une config soignée : signing des commits, aliases, delta comme pager.

Certifications #

Certified Kubernetes Administrator (CKA) — The Linux Foundation & CNCF · Mars 2026

Parcours CKAD (Certified Kubernetes Application Developer) en cours.
